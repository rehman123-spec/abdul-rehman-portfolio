## 📂 Highlighted Projects

*   **MediRead &mdash; Final Year Project (Assistive IoT)**: An assistive technology system designed to help visually impaired and disabled users identify medications. Using a **Raspberry Pi** and **Camera Module**, the system detects medicine bottles using **YOLO**, runs pre-filtering on **OpenCV**, parses the text via **Tesseract OCR**, and outputs synthesized speech warnings and instructions through a **speaker (TTS)**.
*   **Job Shop Scheduling PSO Optimizer (Interactive Web App)**: A browser-based scheduling solver running **Particle Swarm Optimization (PSO)**. Features interactive Gantt charts and Canvas-based convergence graphs. (Accessible locally at `./Job-Shop-Sheduling/index.html`).

## 🚀 How to Host This Online (For Interviewers to See)

Follow these simple steps to put this portfolio live on GitHub so interviewers can see it:

### Step 1: Install Git
If you don't have Git installed, download and install it here:
*   [Download Git for Windows](https://git-scm.com/download/win)

### Step 2: Create a GitHub Repository
1. Log in to [GitHub](https://github.com/).
2. Create a new repository named exactly `rao-abdul-rehman-portfolio`.
3. Keep the repository **Public** (so anyone can view your page).
4. **Do not** add a README, license, or `.gitignore` (we already have a README here!).

### Step 3: Push This Code to GitHub
Open your terminal (PowerShell or Git Bash) inside the project folder `rao-abdul-rehman-portfolio` and run these commands:

```bash
# 1. Initialize local Git repository
git init

# 2. Add all portfolio & project files
git add .

# 3. Create your initial commit
git commit -m "Initial commit: Portfolio with JSSP Project"

# 4. Rename main branch
git branch -M main

# 5. Link to your online GitHub repository
# (Replace <YOUR-GITHUB-USERNAME> with your actual GitHub username)
git remote add origin https://github.com/<YOUR-GITHUB-USERNAME>/rao-abdul-rehman-portfolio.git

# 6. Push your files to GitHub
git push -u origin main
```

### Step 4: Turn on GitHub Pages (Make the Site Live)
Once your code is pushed to GitHub:
1. Go to your repository settings page on GitHub.
2. On the left sidebar, click on **Pages**.
3. Under **Build and deployment**, change the Source to **Deploy from a branch**.
4. Set the branch to **`main`** and folder to **`/ (root)`**, then click **Save**.
5. Wait 1–2 minutes. GitHub will give you a link (e.g., `https://<YOUR-GITHUB-USERNAME>.github.io/rao-abdul-rehman-portfolio/`). 

You can now copy that link and paste it into your resume / CV! Interviewers will be able to load your portfolio and run your JSSP solver online.
